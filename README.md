# html-char-encode-tool
tr karakterlerini html tipine çevirerek her türlü serverda düzgün çıkmasını sağlar 
Tr karakter sorunu yaşamazsınız
('ı', '&#x131;'),
            ('İ', '&#x130;'),
            ('Ö', '&#xD6;'),
            ('ö', '&#xF6;'),
            ('ü', '&#xFC;'),
            ('Ü', '&#xDC;'),
            ('ğ', '&#x11F;'),
            ('Ğ', '&#x11E;'),
            ('ç', '&#xE7;'),
            ('Ç', '&#xC7;'),
            ('ş', '&#x15F;'),
            ('Ş', '&#x15E;'),
            ('â', '&#xE2;'),
            ('Â', '&#xC2;')
gibi dönüştürür sizde bu karakterleri arttırabilirsiniz.
